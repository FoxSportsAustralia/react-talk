What can we adopt from React Js?

  • As developers, we should look towards patterns and adopting what is useful.
  • We should be aiming to make one project with multiple repositories easily understandable and searchable all the way through.
  • For example, the WordPress migration.
  • At Fox Sports, we adopted the components pattern and are now using it across our projects and repositories.
  • This doesn't stop at the just the front end!
